about=Plugin QGIS pour visualiser les données météorologiques Météo-France directement dans QGIS.

  Fonctionnalités v1.2 :
  - Interface guidée en 3 étapes (Territoire → Variable → Options)
  - Sélection visuelle du territoire par cartes cliquables
  - Support AROME métropole (1,3 km) et AROME Outre-Mer (2,5 km)
  - Territoires OM : Antilles, Guyane, Réunion/Mayotte, Nouvelle-Calédonie, Polynésie
  - Variables AROME-OM individualisées avec nom français (Température, Précipitations…)
  - Paquets GRIB2 décodés automatiquement via cfgrib
  - Couches raster GeoTIFF + points GeoPackage (option)
  - Navigateur temporel multi-horizons (onglet Horizons)
  - Bulletins de vigilance Météo-France par département
  - Authentification token / api_key / application_id
